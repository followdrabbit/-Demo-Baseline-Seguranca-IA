
Controles enriquecidos

### Ativar S3 Block Public Access
   - **Controle ID:** amazon_s3_001
   - **Racional:** Impede o acesso público a todos os buckets e objetos armazenados no S3, evitando a exposição acidental de dados sensíveis.
   - **Riscos Mitigados:** Acesso não autorizado e vazamento de dados.
   - **Auditoria:** No AWS Management Console, verifique se o bloqueio de acesso público está ativado em todas as configurações dos buckets S3.
   - **Remediação:** No console do S3, selecione o bucket, vá para a aba 'Permissions', e ative 'Block all public access'.
   - **Referências:** Amazon S3 Security and Data Protection, AWS repost How can I secure the files in my Amazon S3 bucket?
